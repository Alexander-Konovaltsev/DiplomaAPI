# Запуск проекта

## 1. Создание `.env`

Создайте `.env` файл на основе `.env.example`:

```bash
cp .env.example .env
```

После этого заполните необходимые переменные окружения.

---

## 2. Запуск контейнеров

В корне проекта выполните команду:

```bash
docker compose up -d
```

---

## 3. Остановка контейнеров

```bash
docker compose down
```
